How to get started? hide
  1. Expand your browser window so that you see all the six video boxes.
  2. The left panel below represents user list including your name. The right panel on top contains various participant videos including yours. The right panel on bottom is for text chat.
  3. Change your name from "User ..." to your real name in the second edit box below.
  4. Select the checkbox before your name to enable your camera.
  5. Select the checkbox before the names of other people in the user list to view their video.
  6. Enter the text chat message in bottom-center corner in this chat room.
  7. If needed, set your room name in the first edit box below, for a private room, instead of a default room name of "public".
  8. If needed, send private message to a particular user starting with @user name: before your message.
